# Coal Mine Reference Asset Attribution

## `fieg-highwall-reference.obj`

- Source: NASA Ames Intelligent Robotics Group, Planetary Pits and Caves Analog Dataset
- Site: Fieg site, Somerset County, Pennsylvania, USA
- Original file: `FiegHighwallVolume.wrl`
- Download page: https://ti.arc.nasa.gov/dataset/caves/
- Downloaded archive: `data/Fieg/FiegHighwallVolume.rar`
- Description from source page: Fieg was a surface coal mine used for early robot testing and mapped with LiDAR in late 2012 through early 2013. The highwall model is a high-resolution crop of layered wall geology.
- Local processing: converted VRML 2.0 mesh to sampled OBJ with `scripts/convert-wrl-to-obj.mjs`.

This asset is included as a real-world visual reference for demo development. It is a surface highwall reference, not a complete underground mine scan. Do not present it as the source of the coal-gas roadway, goaf, drainage, or robot traversal topology.
