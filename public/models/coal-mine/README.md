# Coal Mine Model Assets

Current bundled reference:

- `fieg-highwall-reference.obj`: NASA Fieg surface coal mine highwall mesh sampled to OBJ.

This is a real geology reference for layered coal wall texture and scale. It is not a full underground mine, roadway, goaf, drainage, or gas-cavity model. The demo coal-gas topology is generated from `src/data/fractureDataGenerator.ts` until a licensed underground mine scan is added.

Future real mine assets must be survey-derived or scan-derived, with license and attribution recorded in `docs/REAL_MINE_ASSET_SOURCES.md` and `ATTRIBUTION.md`.
